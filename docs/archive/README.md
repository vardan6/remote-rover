# Documentation Archive

This directory stores non-authoritative and historical documentation snapshots.

Archive rules:
- archived files are for traceability, not current implementation guidance
- current architecture, status, and plans live under `docs/`
- archived files should include clear replacement pointers where applicable

Top-level groups:
- `root/`: former root-level status/plan snapshots
- `3d-env/`: simulator-side scratch and historical notes
- `gcs_server/`: legacy GCS notes superseded by `docs/gcs_server/`
- `rover-sim-next/`: legacy planning comments superseded by `docs/simulation-platform-plan.md` and `docs/rover-sim-next-phase-1-checklist.md`
